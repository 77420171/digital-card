# 个人数字名片页

## 1. 作品名称
个人数字名片页（Personal Digital Business Card）

## 2. 作品简介
本作品是一个基于HTML+CSS实现的个人数字名片网页，用于展示个人基本信息、技能、兴趣等内容，是一个结构清晰、样式美观、可公开访问的个人作品页面。

## 3. 页面主要内容说明
- **基本信息**：姓名、学号、专业、联系方式占位、个性签名
- **自我介绍**：不少于100字的个人介绍
- **兴趣标签**：前端开发、UI/UX设计、技术阅读、开源项目（≥3个）
- **技能标签**：HTML/CSS、JavaScript、Vue.js、Git/GitHub、Figma（≥3个）
- **个人形象**：本人照片/卡通形象
- **外部链接**：GitHub主页、Gitee主页（≥1个）
- **样式排版**：完整CSS样式，响应式适配，整齐美观

## 4. 文件结构说明
├── index.html # 网页主文件
├── style.css # 样式文件
├── images/ # 图片资源文件夹
│ └── avatar.jpg # 个人头像 / 形象图
└── README.md # 项目说明文档

## 5. 页面公开访问方式
本项目通过 Gitee 部署，公开访问链接：
`https://gitee.com/ll77ii/digital-card/`


## 6. 体现“软件作品”而非“单个页面”的说明
本作品不仅是一个静态HTML页面，而是一个完整的软件作品：
1.  **模块化结构**：将HTML结构与CSS样式分离，代码可维护性强，符合前端开发规范
2.  **响应式设计**：适配电脑、手机等多端设备，具备良好的用户体验
3.  **版本控制**：通过Git进行版本管理，完整的项目结构与文档说明
4.  **可扩展性**：可轻松新增内容、修改样式，具备迭代优化的能力
